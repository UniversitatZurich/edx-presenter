Available content elements

pdf, html, text, file, video, source

Accepted sources

  • local direcory
  • remote GIT repository: Any valid GIT url
  • remote compressed directories (zip, tar.gz), which are accessible using http/https

Options

Please check: 

./edx-presenter.py --help

Examples

Process the project inside the directory ~/python/project1, which contains the groups.yaml file: 

./edx-presenter.py ~/python/project1
A file called to_submit.tar.gz has been created, which can be uploaded to edx. If you want some debug information: 
./edx-presenter.py -v ~/python/project1
Even more debug information: 
./edx-presenter.py -v ~/python/project1

Remark

- All files are expected to be UTF-8 encoded.
